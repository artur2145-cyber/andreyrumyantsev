# ANDREY RUMYANTSEV — сайт персонального тренера

Тёмная одностраничная visit‑карта персонального тренера (бодибилдер, выступающий
спортсмен). Сделана на **Vite + React + TypeScript** в стиле исходного Framer‑референса:
тёмная тема, акцент rust `#AB4617`, заголовки **Clash Display** заглавными, тело — **Inter**.

Языки: **RUS / EST / ENG** (переключатель слева от кнопки «Связаться»).

## Секции

1. **Программы** — направления подготовки (сила, выносливость, масса, питание). Данные — в `src/data/programs.ts`.
2. **AI‑агент** — чат‑виджет, обращается к `api/chat.ts` (прокси к Claude API).
3. **Запись** — собственный календарь бронирования (`src/components/Calendar.tsx`).
4. **Тарифы** — 3 тарифа (разовая / абонемент / персональная программа). Данные — в `src/data/pricing.ts`.
5. **Футер** — логотип, навигация, соцсети, контакты, копирайт.

Меню‑ссылки плавно скроллят к секциям; адаптив на **810px** (планшет, бургер‑меню) и **390px** (телефон).

## Запуск локально

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # сборка в dist/
```

> Чат AI в `npm run dev` показывает заглушку «не подключён», потому что Vite не
> запускает серверную функцию `api/chat.ts`. Чтобы проверить чат локально —
> используйте `vercel dev` (см. ниже) или просто задеплойте на Vercel.

## Контент (аналог CMS)

Всё списочное содержимое лежит в `src/data/` и переведено на 3 языка прямо в файле
(поля `{ ru, et, en }`):

- `src/data/programs.ts` — карточки программ
- `src/data/pricing.ts` — тарифы + контакты/соцсети (`contactInfo`)

UI‑тексты (меню, заголовки секций, подписи) — в `src/i18n/ru.ts`, `et.ts`, `en.ts`.
Цвета и типографика — токены в `src/index.css` (`:root`).

## AI‑ассистент

`api/chat.ts` — serverless‑функция Vercel. Читает ключ из переменной окружения
`ANTHROPIC_API_KEY`. Пока ключа нет — чат отвечает дружелюбной заглушкой.

1. Vercel → **Settings → Environment Variables** → добавить `ANTHROPIC_API_KEY`.
2. (необязательно) `ANTHROPIC_MODEL` — по умолчанию `claude-opus-4-8`,
   можно `claude-haiku-4-5` (дешевле/быстрее).
3. Redeploy.

Виджет настраивается через проп `config` (`endpoint`) — аналог property controls;
подписи берутся из i18n, цвета — из токенов.

## Деплой на Vercel

Самый простой путь:

1. Запушить репозиторий на GitHub.
2. На [vercel.com](https://vercel.com) → **Add New → Project** → импортировать
   `artur2145-cyber/andreyrumyantsev`.
3. Framework определится как **Vite** автоматически (`vercel.json` уже настроен:
   SPA‑rewrites + `/api`).
4. Добавить `ANTHROPIC_API_KEY` (опционально сейчас) → **Deploy**.

Или через CLI: `npm i -g vercel && vercel` (потребуется вход в аккаунт Vercel).
